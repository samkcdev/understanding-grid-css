Learning CSS Grid

CSS grid is declared on the parent element

          .parent{
                display:grid
          }

Its always the direct Descendant that turns into grid

For creating column based layout, we give grid-template-columns: 200px 200px this code will create a two column layout 200px wide.

For creating row based layout, we give grid-template-rows: 100px 200px this code will create a two row layout where the first row will be 100px height and second row will be 200px and third row take the content height.
